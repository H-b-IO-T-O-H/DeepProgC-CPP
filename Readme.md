# Углубленный курс по C/C++

### Условие ДЗ №1
```
Создать структуру для хранения информации о небоскребах мира: количестве этажей, общей высоте, высоте шпиля и назначении здания, 
а также регионе расположения. Составить с ее использованием программу вывода информации обо всех зданиях с группировкой сначала 
по назначению (офисные, гостиничные и т.д.), затем — по региону расположения (Европа, Ближний Восток, Юго-Восточная Азия и т.д.).
```

### Условие ДЗ №2
```
В вашем распоряжении — два вектора из 100 млн. чисел каждый. Определите, является ли один из них точной копией другого, записанной в обратном порядке.
Составьте наивный алгоритм решения задачи, а затем реализуйте параллельный алгоритм с использованием нескольких процессов с учетом оптимизации работы с кэш-памятью.
```

Выполненное дз можно найти в ветках.
